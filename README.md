# Spark GTFS

[Apache Spark](https://spark.apache.org) library for handling [General Transit Feed Specification](https://developers.google.com/transit/gtfs/reference/) 
data files.

## Dependencies
- Spark 2.2.0
- Scala 2.11.12
- sbt 0.13.16
